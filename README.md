# SQL Cyberchase

A collection of SQL solutions for **CS50's Introduction to Databases with SQL - Problem Set 0: Cyberchase**.

This repository contains my practice work while learning SQL fundamentals using SQLite. The exercises involve querying a database of Cyberchase episodes and solving problems using SQL queries.

## About

The Cyberchase problem set focuses on building a foundation in relational databases and SQL.

Topics practiced:

- SELECT statements
- WHERE filtering
- ORDER BY sorting
- LIMIT
- Aggregate functions
- GROUP BY
- LIKE pattern matching
- Data analysis with SQL

## Running the Queries

Install SQLite, then open the database:

```bash
sqlite3 cyberchase.db
```

Run any SQL solution:

```sql
.read 1.sql
```
